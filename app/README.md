# Hydration Hero

A tiny desktop companion from Strawvarie. A pixel-art buddy lives in your menu
bar / system tray and gently reminds you to sip from your tumbler through the
day. Fully offline — no accounts, no tracking, nothing leaves your computer.

Built with **Tauri v2** (Rust core + TypeScript UI). macOS and Windows from one
codebase.

---

## Run it locally (dev)

You need two things installed once:

1. **Node.js 18+** — https://nodejs.org
2. **Rust** — https://rustup.rs (`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`)

Then, from this `app/` folder:

```bash
npm install
npm run tauri dev
```

The first run compiles the Rust core and takes a few minutes. After that it's
fast. A window opens with the onboarding flow; once you finish, the app lives in
your menu bar (macOS, top-right) or system tray (Windows, bottom-right).

**To see a reminder immediately:** finish onboarding, then click the tray icon →
**Remind me now**. (Or use **Preview a reminder** in Settings.)

> Tip: the default reminder interval is 45 minutes. To watch the real timer fire
> quickly, set the interval to its minimum in Settings, or just use "Remind me
> now".

## Build installers

```bash
npm run tauri build
```

Outputs land in `src-tauri/target/release/bundle/`:

- **macOS:** `.app` and `.dmg` (universal build needs the Rust targets — see below)
- **Windows:** `.exe` (NSIS installer) and `.msi`

> **Signing note:** these builds are **unsigned** (no paid certificates yet).
> On first launch macOS will need Right-click → Open (or System Settings →
> Privacy & Security → Open Anyway on macOS 15+), and Windows may show a
> SmartScreen "More info → Run anyway" prompt. See the main plan for the
> recommended distribution path (Microsoft Store for Windows; Apple Developer
> Program for a clean Mac install).

For a macOS universal binary (Intel + Apple Silicon):

```bash
rustup target add x86_64-apple-darwin aarch64-apple-darwin
npm run tauri build -- --target universal-apple-darwin
```

## Regenerate characters / icons

The three built-in characters and all app icons are generated by a script
(clean placeholders — real art gets commissioned later, see
`docs/character-brief.md`). To re-run it you need Python with Pillow:

```bash
python3 -m pip install Pillow
python3 scripts/generate_assets.py
```

Adding a new character = drop a folder in `public/characters/<id>/` (an
`atlas.png` + `manifest.json`) and add one line to `src/shared/characters.ts`.

## What's intentionally simple in V1

These are deliberate scope choices (kept lean on purpose), not missing work:

- **Compact corner overlay** instead of per-pixel screen-wide click-through. The
  overlay appears in a corner and never steals typing focus.
- **Active-hours + easy pause** instead of automatic meeting/full-screen
  detection. Pause from the tray in one click; set quiet hours in Settings.
- **Manual update check** (opens GitHub Releases) instead of a background
  auto-updater — no signing/hosting set up yet.
- **English only**, three built-in characters, reminders only (no water logging,
  goals, or streaks).

## Settings & data

Stored as one JSON file, on-device only:

- **macOS:** `~/Library/Application Support/in.strawvarie.hydrationhero/settings.json`
- **Windows:** `%APPDATA%\in.strawvarie.hydrationhero\settings.json`

Deleting that file resets the app to first-run.
